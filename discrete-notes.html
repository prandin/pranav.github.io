Discrete Mathematics [Lecture Notes]

Course Structure:

Propositional Logic

  1. Check whether the following sets of propositions are consistent or not:
    1. {NOT(A) AND B, NOT(B), A}
    2. {IF A THEN B, IF A THEN C, IF B THEN NOT(C), A}
  2. Consider the following proposition: ∃xP(x) ∨ ∀ ∃xQ(x). Is the value of x referred by both P and Q same? [No!]
  3. Rewrite the proposition "B if A" in terms of (a) necessity, and (b) sufficiency. [(a) B is necessary for A, (b) A is sufficent for B]

Graph Theory

Graphs - degree, order, simple graphs, Havel-Hakimi's Theorem for Simple Graphs

  1. What is the degree a vertex Q that is connected to itself and to another vertex P via only one edge?
  2. A graph G has the degree sequence (4, 3, 2, 2, 1). Is G a simple graph?

Cycle Graphs; Bipartite Graphs;

  1. Find the maximum number of simple graphs that can be created using labelled vertices {v1, v2, ... , vn}?
  2. Show that the cycle graph C4 is bipartite.
  3. Find the maximum number of edges possible in a bipartite graph with n vertices. [Max. occurs when both sets have n/2 vertices]
