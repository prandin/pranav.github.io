Virtual Memory

  1. A process has 1024 page tables in memory. Its corresponding page table is also stored in memory. The overhead for reading a word from the page table is 5 ms. To reduce this overhead, the system has a TLB holding 32 entries and can perform a complete search in 1 ms. Find the minimum hit rate (in %) requird to reduce the mean overhead of accessing the page table to at most 2 ms.
  2. An instruction takes a time of 5 ms and a page fault takes an additional time of 2 ms. If on average a page fault occurs at every 100th instruction, find the effective instruction time.
  3. A computer system implemnts 8 KB pages and a 32-bit physical address space. Each page table entry contains a valid bit and the translation. If the maximum size of the page table of a process is 20 megabytes, then find the length of the virtual address supported by the system (in bits).
  4. Suppose a system uses SJF scheduling and it is estimated that the exponential average of the measured length of previous CPU burst is 0.25. If the initial value of the predicted CPU burst time is 4 units, find the predicted time (in units) for the 4th CPU burst for a process with burst times of 4 units, 12 units and 8 units respectively.
  5. Assume that we have a demand-paged memory. The page table is held in registers. It takes 8 milliseconds to service a page fault if an empty frame is available or if the replaced page is not modified and 20 milliseconds if the replaced page is modified. Memory-access time is 100 nanoseconds. Assume that the page to be replaced is modified 70 percent of the time. What is the maximum acceptable page-fault rate for an effective access time of no more than 200 nanoseconds? [Hint: Consider each case and their probability: to find a page in main memory, page faulted - space available/ un-modified frame + all modified frames]
